# 🚉 Jaipur Station Navigation System

A smart navigation web app for Jaipur Railway Station.

## 🌐 Live Demo

👉 [Click here to view the project](https://jaipur-station-navigation.vercel.app/login)

## ⚠️ Note
Frontend-only deployment for demo purposes.

## 🔥 Features

* 🔐 User Authentication (JWT)
* 🗺️ Interactive Google Maps
* 📍 Facility Navigation with live directions
* 🚆 Train Schedule Module
* 🌍 Multilingual Support (English/Hindi)

## 🛠 Tech Stack

* React.js
* Node.js + Express
* MySQL
* Google Maps API

## 🚀 How to Run

### Frontend

cd frontend
npm install
npm start

### Backend

cd backend
npm install
node server.js
